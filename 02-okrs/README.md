# 02-okrs

Objectives and key results for the Jewell Life OS. One file per half-year, named
`OKRS-<year>-<half>.md`. The current file is the only live one; superseded files stay as the
record, never deleted.

## The discipline

- 3 to 4 objectives per half. Each objective has 3 to 5 measurable key results.
- Every key result has one owner and a number or a date. No vague verbs.
- Reviewed at the Friday close (movement only), scored monthly 0.0–1.0 at the OS scorecard,
  reset each half. A score of 0.7 is success; 1.0 every time means the target was soft.
- OKRs are strategy. Changing an objective mid-half is Clent's call, logged in the decision
  log. Key-result tuning is Tier 2.
- Work OKRs live here. Personal OKRs live in the private repo
  (`clent-jewell-personal/OKRS-PERSONAL.md`) and are never summarised outward.
