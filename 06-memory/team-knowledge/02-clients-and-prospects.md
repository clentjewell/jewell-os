# Clients & Prospects Directory

Every client folder in Drive follows the standard 3D shell:
`00 Admin · 01 Discover · 02 Design · 03 Deploy · 04 Deepen + Intelligence ·
05 Blueprint Extraction · (06 Source + Working + Imports) · 99 Archive`.
Each client has (or gets) a private `#client-<name>` Slack channel and a KH00
Knowledge Hub index doc as the front door. New client environments are
provisioned by Sam via `scaffold-client.js` (Drive shell + Asana project +
Slack channel).
