# Systems of record

The short source-of-truth table for this kit. Full detail and the complete map live in
`../../OPERATING-SYSTEM.md` Section 3 and `../../00-governance/source-of-truth-map.md`. This file
exists so a pack in this kit can link to a one-line answer without restating the full map.

## The table

| Object | Home |
| --- | --- |
| Files | Google Drive |
| Actions | Asana |
| Decisions | The decision log |
| Rules and AI instructions | `jewell-os` |
| Secrets | Approved secret store |
| Calendar | Google Calendar |
| Finance | Xero |

## The rule

One source of truth per object. Never a second master. If a pack in this kit needs to record
something not on this table, check `../../OPERATING-SYSTEM.md` Section 3 before creating a new
home for it.

**Next:** see `ai-authority-matrix.md` for what an AI agent may do with each of these on its own.
