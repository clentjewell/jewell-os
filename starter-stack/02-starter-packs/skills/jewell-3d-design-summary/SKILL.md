# Jewell 3D Design summary

Turn an approved Discover pack into a gated set of Design drafts.

## When to use

Once the Discover pack for the engagement is approved and the engagement owner is ready to move
into Design.

## Inputs

- The approved Discover pack from `01_Discover`.
- Any Design-stage direction the engagement owner has already given (positioning steer, format
  preferences, prior Design decisions in the log).

## Outputs

- A gated Design draft set in `02_Design`: strategy, positioning, messaging, offers, and any
  other Design artifacts the engagement calls for (site, portal, campaign, workflow, content,
  measurement plan), each clearly marked DRAFT.

## Steps

1. Read the approved Discover pack. Do not start Design work from an un-approved Discover pack.
2. Draft each Design artifact the engagement needs, grounded in Discover, not invented from
   scratch.
3. Mark every file and every section clearly as DRAFT.
4. Place the draft set under `02_Design`, in a drafts subfolder, separate from anything approved.
5. **Approval gate.** The engagement owner reviews the draft set. Only the engagement owner
   promotes a draft to `05_Approved`. A draft is never client-visible before that promotion.

## Guardrails

- Must only draw from an approved Discover pack, never an un-approved one.
- Must never present a draft as approved. Every Design output stays marked DRAFT until promoted.
- Promotion to `05_Approved` is a Tier 2 client-facing action (it can become client-visible),
  performed only by the engagement owner, never by the skill itself.

## Where the output is filed

Drafts to `02_Design` (drafts subfolder). Approved work is promoted by the engagement owner to
`05_Approved`.

## Example

Acme Co's Discover pack is approved. The skill drafts positioning, messaging and a measurement
plan into `02_Design`, each headed "DRAFT — not for client view". The engagement owner reviews,
edits the messaging draft, and promotes the approved version to `05_Approved`.

## Success measure

Design drafts pass the engagement owner's first review with only light edits needed, not a
rebuild.
